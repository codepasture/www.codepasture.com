# www.codepasture.com

Marketing website for Code Pasture LLC, a software development consultancy based in Michigan.

## Tech Stack

- **Framework**: SvelteKit 2 / Svelte 5
- **Styling**: CSS Variables + Svelte Scoped Styles
- **Build**: Vite 7
- **Deployment**: Netlify (static adapter)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
